VS常用快捷方式

发表于 2019-03-18

以下是一些Visual Studio 常用的快捷方式

窗口快捷键

Ctrl+W,S: 解决方案管理器 （解决方案是Solution）
Ctrl+W,E: 错误列表 （错误是Error）
Ctrl+W,O: 输出视图 （输出Output ）
Ctrl+D,B: 断点窗口 （断点 breakpoint）
Ctrl+D,I: 即时窗口 （即时immediately ）

项目功能快捷键

F7: 查看代码
Ctrl+Shift+N: 新建项目
Ctrl+Shift+O:打开项目
CTRL + F4关闭文档窗口
CTRL + SHIFT + B:生成解决方案
Ctrl+Shift+S: 全部保存
Ctrl+Shift+A: 新建项
Shift+Alt+C: 新建类

查找相关快捷键

Ctrl+F: 查找
F3: 查找下一个
Shift+F3: 查找上一个
Ctrl+H: 替换

代码快捷键

Ctrl+E,D（ctrl+k,d）格式化全部代码
Ctrl+K,C: 注释选定内容
Ctrl+K,U: 取消选定注释内容
Ctrl+K,S: 插入外侧代码（例如region）
CTRL + M:折叠或展开当前方法
CTRL + M, CTRL + O 折叠一个方法
CTRL + M, CTRL + L展开所有方法
插入自动属性：输入prop后按两下tab
插入依赖属性：输入propdp后按两下tab

调试快捷键

F5: 启动调试
Ctrl+F5: 开始执行(不调试)
Shift+F5: 停止调试
Ctrl+Shift+F5: 重启调试
Ctrl+F9: 启用/停止断点
Ctrl+Shift+F9: 删除全部断点
F10: 逐过程
Ctrl+F10: 运行到光标处
F11: 逐语句

编辑快捷键

Shift+Alt+Enter: 切换全屏编辑
F12: 转到所调用过程或变量的定义
Alt+F12: 查找符号
shift+f12:查找所有引用
Ctrl+U: 全部变为小写
Ctrl+Shift+U: 全部变为大写
Ctrl+Shift+L: 删除当前行（Ctrl+X）
Ctrl+E,S: 查看空白
Ctrl+E,W: 自动换行
Ctrl+G: 转到指定行
Shift+Alt+箭头键: 选择矩形文本
Alt+鼠标左按钮: 选择矩形文本
CTRL + DELETE删除至词尾
CTRL + BACKSPACE删除至词头
CTRL + SHIFT + PAGE UP选择至本页前面
CTRL + SHIFT + PAGE DOWN选择至本页后面
CTRL + PAGE DOWN光标定位到窗口上方
CTRL + PAGE UP光标定位到窗口下方
CTRL + END文档定位到最后
CTRL + HOME文档定位到最前
按两下tab快速插入代码段
ctrl+减号：回退到光标上一次的位置
